# 개인정보 처리방침

`AI와 함께하는 수과정페 네컷`은 2026. 진주시와 함께하는 경남과학고등학교 수학, 과학, 정보 페스티벌에서 AI 네컷사진 생성 및 이메일 발송을 위해 필요한 최소한의 개인정보만 처리합니다.

본 문서는 행사 운영자가 현장 안내문, 동의 화면, 운영 절차를 정리하기 위한 기준입니다. 실제 행사 운영 전에는 담당 교사 또는 학교 개인정보 보호 담당자의 최종 검토를 거쳐 게시해야 합니다.

- 시행일: 2026-05-15
- 운영 주체: 경남과학고등학교 이음(IEUM) 동아리
- 부스명: AI와 함께하는 수과정페 네컷
- 문의: 행사 현장 운영요원에게 문의하거나, 운영 전 지정되는 담당자 연락처를 본 문서에 기입합니다.

## 1. 처리하는 개인정보 항목

### 필수 항목

| 항목 | 내용 |
| --- | --- |
| 촬영 사진 | AI 배경 추천용 사진 1장, 최종 네컷 제작을 위해 사용자가 선택한 사진 |
| 생성 결과 이미지 | AI 배경과 선택 사진을 합성한 최종 네컷 이미지 |
| 이메일 주소 | 사용자가 이메일 수신을 선택한 경우 입력한 수신 이메일 주소 |
| 동의 기록 | 필수 개인정보 수집 및 이용 동의 여부, 선택 사진 저장 동의 여부 |
| 운영 기록 | 완성 시간, 선택한 배경 특징, AI 사용량 및 비용, 메일 전송 성공/실패/건너뜀 상태 |

### 선택 항목

| 항목 | 내용 |
| --- | --- |
| 완성 사진 저장 동의 | 행사 홍보 및 결과 전시를 위한 완성 사진 저장 동의 여부 |
| 선택 저장 완성 사진 | 사용자가 선택 저장에 동의했고, 운영 환경에서 사진 아카이브가 켜져 있을 때만 저장되는 완성 사진 |

기본 설정에서는 관리자 기록에 원본 사진, 완성 사진 이미지, 이메일 주소를 저장하지 않습니다.

## 2. 개인정보 처리 목적

수집한 개인정보는 다음 목적에만 사용합니다.

1. AI 네컷사진 생성
2. AI 배경 추천 및 배경 이미지 생성
3. 크로마키 기반 인물 분리, 얼굴 보정, 최종 네컷 합성
4. 사용자가 입력한 이메일 주소로 완성 사진 발송
5. 행사 운영 중 오류 대응 및 전송 실패 확인
6. 사용자가 별도로 동의한 경우 행사 홍보 및 결과 전시

위 목적 외 용도로 사용하지 않으며, 목적이 변경될 경우 별도 동의를 받습니다.

## 3. 개인정보 보유 및 이용 기간

| 구분 | 보유 기간 |
| --- | --- |
| 정상 완료 세션의 촬영 사진, 생성 결과 이미지, 이메일 주소 | 이메일 발송 또는 메일 건너뛰기 완료 후 즉시 삭제 |
| 오류 또는 미완료 세션 파일 | 오류 대응을 위해 최대 24시간 보관 후 삭제 |
| 기본 관리자 기록 | 완성 시간, 선택 키워드, AI 비용, 메일 상태만 최대 24시간 보관 후 삭제 |
| 선택 저장 완성 사진 | 선택 저장 동의가 있고 사진 아카이브가 켜진 경우에만 저장하며, 행사 홍보 및 결과 전시 목적 달성 후 삭제합니다. 운영 전 구체적인 삭제 예정일을 정해 현장에 안내해야 합니다. |

운영 환경에서는 `SESSION_TTL_MINUTES`, `ADMIN_ARCHIVE_TTL_HOURS`, `ADMIN_ARCHIVE_ENABLED`, `ADMIN_ARCHIVE_STORE_IMAGES` 설정을 통해 보관 범위를 제한합니다.

## 4. 외부 서비스 이용

서비스 제공을 위해 다음 외부 API를 사용합니다.

| 서비스 | 이용 목적 | 전송될 수 있는 정보 |
| --- | --- | --- |
| OpenAI API | 포즈 분석, 배경 이미지 생성 | AI 배경 추천용으로 축소된 분석 사진 1장, 선택한 배경 특징, 배경 생성 프롬프트 |
| Brevo Email API | 완성 사진 이메일 발송 | 수신 이메일 주소, 최종 네컷 이미지 첨부파일 |

최종 네컷 제작에 사용되는 선택 사진 4장의 원본은 OpenAI 배경 생성 API로 보내지 않습니다. MediaPipe 기반 크로마키/인물 분리와 얼굴 보정은 브라우저 또는 로컬 서버에서 처리합니다.

OpenAI와 Brevo는 해외 사업자가 제공하는 서비스이므로 API 처리 과정에서 국외 서버로 정보가 전송될 수 있습니다. 행사 운영 전 각 서비스의 최신 개인정보 처리 조건, 데이터 처리 약관, 보관 정책을 확인해야 합니다.

## 5. 동의 거부 및 서비스 이용 제한

필수 개인정보 수집 및 이용에 동의하지 않으면 이메일 발송형 AI 네컷사진 서비스를 이용할 수 없습니다.

선택 사진 저장 동의는 거부할 수 있으며, 선택 동의를 하지 않아도 촬영, 합성, 이메일 발송 서비스는 이용할 수 있습니다.

## 6. 만 14세 미만 참가자

만 14세 미만 참가자는 보호자 또는 인솔자의 동의가 필요합니다. 현장 운영자는 만 14세 미만으로 보이는 참가자가 단독으로 이용하지 않도록 안내해야 합니다.

## 7. 개인정보의 파기

보유 기간이 끝나거나 처리 목적이 달성되면 해당 개인정보를 지체 없이 삭제합니다.

- 세션 임시 파일은 이메일 발송 또는 건너뛰기 완료 후 삭제합니다.
- 오류 또는 미완료 세션은 최대 24시간 이내 정리합니다.
- 관리자 기록은 사진과 이메일 주소를 제외한 운영 통계 중심으로 보관합니다.
- 선택 저장 사진은 운영자가 정한 게시 또는 전시 기간이 끝나면 삭제합니다.

## 8. 개인정보 보호 조치

운영자는 다음 조치를 적용합니다.

1. 관리자 페이지 PIN 보호
2. 관리자 API 인증 쿠키 확인
3. 정리 API의 `CRON_SECRET` 보호
4. 세션별 임시 저장소 사용 및 완료 후 삭제
5. 업로드 이미지 용량 및 픽셀 수 제한
6. 주요 API rate limit 적용
7. 이벤트/에러 로그에 이메일 주소와 원본 사진 저장 금지
8. 기본 관리자 아카이브 비활성화
9. 운영 전 `/admin/health` 점검

## 9. 정보주체의 권리

참가자는 본인의 개인정보에 대해 열람, 정정, 삭제, 처리정지를 요청할 수 있습니다.

다만 본 서비스는 이메일 발송 또는 건너뛰기 완료 후 세션 파일을 즉시 삭제하도록 설계되어 있어, 이미 삭제된 정보는 열람 또는 정정이 어려울 수 있습니다. 오류 대응 기간 중 보관된 세션 정보는 운영자에게 삭제를 요청할 수 있습니다.

## 11. 처리방침 변경

이 개인정보 처리방침은 행사 운영 방식, 외부 서비스, 보관 정책이 변경될 경우 업데이트합니다. 중요한 변경이 있는 경우 행사 현장 안내문과 동의 화면에 반영합니다.
