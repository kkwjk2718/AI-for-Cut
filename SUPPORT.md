# 지원 받기

## 버그 제보

GitHub Issues의 버그 템플릿을 사용해 주세요. 가능하면 아래 내용을 포함합니다.

- 재현 단계
- 기대한 동작
- 실제 동작
- 브라우저와 실행 환경
- 콘솔 오류 또는 서버 로그

참가자 사진, 이메일 주소, API 키는 첨부하지 않습니다.

## 기능 제안

기능 제안 템플릿을 사용해 목적, 사용자 흐름, 운영상 이점을 적어 주세요. 행사 당일 안정성에 영향을 주는 변경은 구현 전에 먼저 논의합니다.

## 운영 문제

행사 운영 전에는 `/admin/health`에서 OpenAI, Brevo, 저장소, 디스크, 브랜드 asset 상태를 확인하세요.

운영 중 메일 전송이나 배경 생성이 실패하면 화면의 오류 문구와 관리자 헬스 체크 결과를 함께 확인합니다.
